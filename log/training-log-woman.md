# Журнал тренировок и самочувствия (для неё)

> Сюда пишем фактику после тренировок: что сделала, с какими весами/повторами, что было легко/тяжело, самочувствие, сон, боли. На основе этого корректируется `../program/current-block-woman.md`.
> Формат свободный — можно просто накидать текстом в новой сессии, Claude разложит и внесёт сюда.

## Шаблон записи
```
### Микроцикл N · Тренировка (День A / День B / День C / День D) · дата
- Основное: <упр> <вес>×<повт>×<подх>, запас ~<RIR>. Ощущение: легко/норм/тяжело.
- Подсобка: <упр> <вес>×<повт> ...
- Самочувствие: сон, энергия, стресс, аппетит.
- Боли/дискомфорт: где, при каком движении.
- Пропуски/замены упражнений: ...
```

## Как я (Claude) реагирую на фидбек
- **«Легко, запас 4+»** → поднять вес/повторы раньше графика.
- **«Не добрала нижний повтор / запас 0»** → снизить вес или откатить неделю.
- **Боль в суставе** → заменить упражнение, снизить нагрузку, флажок в план.
- **Плохой сон/энергия несколько тренировок** → снять объём, возможно ранняя разгрузка.
- Любое изменение фиксирую в «Журнал изменений плана» внутри current-block-woman.md.
