# Соглашения по составлению назначений

## Разминка со штангой: веса кратны 5 кг на сторону, только добавление блинов

Для упражнений со штангой (в первую очередь ягодичный мост) разминочные подходы задаются так, чтобы вес каждого следующего подхода получался добавлением блинов к предыдущему, без снятия. Нагрузка на каждую сторону грифа кратна 5 кг (гриф → +5/сторона → +10/сторона → …). Не назначать промежуточные значения вроде 35 кг (= 7.5 кг/сторона), требующие снять блин перед рабочим подходом.

Пример для моста с рабочими 50 кг (гриф 20 кг): 20 → 40 (+10/сторона) → 50 (+15/сторона).
