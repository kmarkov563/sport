# Соглашения по составлению назначений

Устойчивые правила формата назначений для этого атлета — в дополнение к разделу 4 `AI_COACH.md`. Растут от запросов атлета; при новом запросе дополнять.

## Разминка со штангой: веса кратны 5 кг на сторону, только добавление блинов

Для упражнений со штангой (в первую очередь **ягодичный мост**) разминочные подходы задаются так, чтобы вес каждого следующего подхода получался **добавлением** блинов к предыдущему, без снятия. Практически: нагрузка на каждую сторону грифа кратна 5 кг (гриф → +5/сторона → +10/сторона → …). Не назначать промежуточные значения вроде 35 кг (= 7.5 кг/сторона), которые заставляют снимать блин перед рабочим подходом.

Пример для моста с рабочими 50 кг (гриф считаем 20 кг): 20 (гриф) → 40 (+10/сторона) → рабочие 50 (+15/сторона).
