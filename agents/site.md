# Сайт: хостинг и проверка

## Хостинг — GitHub Pages

Сайт публикуется как **GitHub Pages** из ветки `main`. Живой адрес: **https://kmarkov563.github.io/sport/**.

Страница отдаётся по HTTPS (не `file://`), поэтому `fetch('log/man/index.json')` и подобные запросы к локальным файлам репозитория работают штатно — данные можно смело держать в отдельных `.json` и подгружать со страницы. Пути в `fetch` — относительные (сайт лежит в подпапке `/sport/`). `index.html` — корневая страница.

## Проверку в браузере делает пользователь

Не проверять изменения в браузере: без `claude-in-chrome`, без локального http-сервера, без скриншотов. После правок `coach.html` / JSON-планов просто описать, что изменено, и остановиться — пользователь сам открывает страницу и сообщает, если что-то не работает.
