# Субагент — статистика микроцикла

Запускается в конце промта (после задач A / B / C), по одному на каждый изменённый микроцикл. На вход — путь к файлу микроцикла (напр. `log/man/0001.json`). Задача: пересчитать и вписать в этот файл сущность `microcycleStats` — сводку нагрузки за цикл. Её показывает `coach.html` внизу страницы; нормы подходов по мышцам и пояснения к цифрам живут в `coach.html`, в JSON их не дублировать.

## Что берём в расчёт

- **Все** сессии файла — и выполненные, и запланированные. Выполнено или нет — неважно: считаем по всем тренировкам микроцикла (у запланированных — по назначенным весам). Стата пишется всегда.
- Упражнение с несколькими `variants` (снаряд на выбор) — считать по **одному** варианту (первому в списке): упражнение выполняется один раз, варианты не суммировать.
- Внутри сессии — только **рабочие** подходы. Разминочный/подводящий исключить: `rir` строкой («разминка»), `note` содержит «разминк»/«подвод», либо нет веса (`weight` пуст или 0) при том, что в упражнении есть весовые подходы.

## Поля `microcycleStats`

- `tonnageKg` — Σ(вес_кг × повторы × подходов). Вес_кг по единице: `kg` ×1, `kg_per_hand` ×2 (две гантели), `kg_per_side` ×2 (обе стороны), `plates` ×5; упражнение без веса (вес тела, базовый вес тренажёра) — 1 кг.
- `avgIntensityPct` — среднее по рабочим подходам (вес / e1RM упражнения в той же сессии) × 100. `e1RM` = максимум по рабочим подходам упражнения (reps ≤ 12) от `weight × (1 + (reps + rir) / 30)`; `rir` отсутствует → 1.5. Подходы без e1RM (у всех reps > 12, либо нет весовой базы) в среднее не входят.
- `hardSets` — число рабочих подходов с `RIR ≤ 3` (`rir` отсутствует → 1.5).
- `hardSetsByMuscle`, `tonnageByMuscleKg` — те же величины в разрезе **основной** мышцы упражнения. Мышцу брать из `agents/muscle-map.md` (не решать самостоятельно) — по разделу нужного атлета. Упражнения там нет — определить основную мышцу, дописать его в карту (при неоднозначности уточнить), затем считать. Оба объекта отсортировать по убыванию значения.

`tonnageKg`, `tonnageByMuscleKg`, `avgIntensityPct` округлять до целого. Если `microcycleStats` уже есть — перезаписать. Формат — как в готовом `log/man/0001.json`.
