# Карта мышц: упражнение → основная мышца

Справочник для субагента статистики (`task-microcycle-stats.md`): по названию упражнения из лога берётся **одна основная** мышечная группа. Карта раздельна по атлетам — одно движение может грузить разное в зависимости от программы (у мужчины болгарские — квадрицепс, у женщины — ягодицы).

Упражнения, которого здесь нет, — определить основную мышцу (по `base/exercises.txt`), **дописать сюда**; при неоднозначности уточнить у пользователя. Ключи групп: `Грудь`, `Спина`, `Плечи`, `Бицепс`, `Трицепс`, `Квадрицепс`, `Бицепс бедра`, `Ягодицы`, `Икры`, `Пресс`, `Задняя цепь`.

- **Бицепс бедра** — Румынская тяга в Смите · Румынская тяга · Румынская тяга (штанга или Смит на выбор) · Сгибание ног в тренажёре

- **Пресс** — Пресс · Планка на локтях · Пресс (упражнение на выбор)
